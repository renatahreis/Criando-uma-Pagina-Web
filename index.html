Apresentação do Curso

(Voltar)

Este é um tópico de apresentação do módulo, que tem finalidade de ensinar a instalar e configurar o ambiente de desenvolvimento, onde podemos ver sobre arquivos HTML, Principais ferramentas utilizadas, Inspetor de Elementos, Estrutura básica do HTML, Tags, Atributos, Listas Ordenadas e Não ordenadas e etc... Tendo por objetico apresentar a base para desenvolvimento de uma pagina WEB.

Estrutura Básica do HTML

(Voltar)

Esta aula é apresentado na prática as estruturas básicas necessárias para criação de uma página HTML.

Falando sobre Tags

(Voltar)

Esta aula fala sobre as tags. O que são tags? São elementos utilizados para informar ao navegador a estrutura do site. Ou seja: quando se escreve um código em HTML, as tags serão interpretadas pelo navegador, produzindo assim a estrutura e o conteúdo visual da página.

Atributos das Tags

(Voltar)

Esta aula é a apresentação dos atributos, elementos para personalizar as tags e modificar suas funcionalidades. Algum dia você quer redimencionar uma imagem ou tabela ou mudar a cor da fonte. Tudo isso é possível com a ajuda de atibutos.

Textos

(Voltar)

Nesta aula foi apresentado as principais tipografias utilizadas para edição de textos. O que é tipografia? Tipografia significa a impressão dos tipos, sendo que tipo é a fonte, ou a letra.

Listas

(Voltar)

Esta aula foi apresentado os dois tipos de listas e como utiliza-las. O que é listas? Lista é um recurso de HTML que permite criarmos tópicos de textos para uma melhor exemplificação de um determinado assunto.

Links

(Voltar)

Nesta aula o professor apresenta a tag de ancorar, utilizada para anexar os links. Como funciona um link? Os links conectam uma página da internet até outra, eles levam o usuário de um local a outro.
